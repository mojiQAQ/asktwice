一、项目概述

项目名称：Ask Twice（再问一遍）
产品形态：Chrome/Edge 浏览器插件

二、技术可行性分析

结论：✅ 技术上完全可行，一个人可以搞定

2.1 核心技术栈评估

① Chrome 插件开发（难度：🟢 低）
• Manifest V3 + Content Script → 成熟技术栈
• 注入到 AI 页面 → 监听 DOM 变化提取回答文本
• MutationObserver 处理动态加载内容
• 预计开发时间：1-2 天搭好框架

② 后端 API 开发（难度：🟢 低）
• Python (FastAPI) 或 Node.js (Express) → 几百行代码
• 核心是 LLM API 调用 + 搜索 API 集成
• 无需复杂数据库设计（初期 PostgreSQL 就够）
• 预计开发时间：2-3 天

③ LLM-as-a-Judge 方案（难度：🟡 中）
• Claim Extraction（声明提取）→ 调用 GPT-4o-mini / Claude
• GEO 检测 → Prompt Engineering 判断推荐偏向
• 利益冲突 → 关键词匹配 + LLM 辅助判断
• 核心挑战：prompt 设计决定准确率
• 预计迭代周期：持续优化，但 V1 可在 1-2 天内完成

④ 搜索来源验证（难度：🟢 低）
• Brave Search API（免费 2000 次/月）或 Google Custom Search API（$5/千次）
• 查询 AI 声明 → 匹配原始来源 → 评估来源权威度
• 实现简单，核心成本在 API 调用费

2.3 MVP 开发计划

3-5 天出 MVP，分三步走：

Day 1-2：Chrome 插件骨架
• 搭建 Manifest V3 框架
• 实现 Content Script 注入（优先支持豆包 + ChatGPT）
• 提取 AI 回答文本 → 发送到后端

Day 3-4：后端 API + LLM 集成
• FastAPI 搭建后端服务
• 集成 GPT-4o-mini API（claim extraction + GEO 判断）
• 集成 Brave Search API（来源验证）
• 返回综合评分

Day 5：前端展示 + 打磨
• 插件覆盖层（overlay）显示可信度评分
• 评分气泡 UI 设计
• 基本的错误处理和加载状态

三、市场可行性分析

结论：✅ 市场窗口期真实存在，但需快速执行

3.1 市场规模

宏观市场
• 中国 AI 搜索月活用户：6 亿+（2025 IDC 数据）
• 中国 GEO 服务市场：2026 年预计突破 180 亿元
• AI 安全/信任市场全球：2025 年约 $50 亿，年增长 40%+

目标市场（TAM → SAM → SOM）
• TAM（总可及市场）：6 亿 AI 搜索月活用户
• SAM（可服务市场）：约 5000 万"高频+高意识"用户
  （经常用 AI 且在意答案质量的知识工作者、学生、研究人员）
• SOM（短期可得市场）：第一年 10-50 万用户（占 SAM 的 0.2-1%）

3.2 需求验证信号

强需求信号 🟢
• 96% 的企业 AI 使用在 IT 监控之外（Aurascape 数据）→ 用户对 AI 信任存疑
• 豆包嵌入抖音商城链接引发用户投诉 → 利益冲突需求真实
• Facticity.AI 获 Tim Draper 投资 + TIME 年度最佳发明 → 资本认可方向
• 学术界 GEO 研究快速升温（Princeton 2024 论文后大量跟进研究）
• AI Agent 安全赛道融资火热（Aurascape $5000万）→ 大主题成立

弱需求信号 🟡
• C 端用户为"AI 答案验证"付费的意愿尚未被充分验证
• 大多数普通用户对"AI 幻觉"缺乏认知
• 315 没有直接曝光 AI 推荐问题 → 公众议题还不够热

四、商业可行性分析

结论：⚠️ 商业化可行但需要探索，C 端付费是最大不确定性

4.1 收入模型

阶段 1：免费获客（0-6 个月）
• 免费版：每天 5 次验证
• 目标：积累 1-5 万用户 + 产品打磨

阶段 2：订阅变现（6-12 个月）
• Pro 版：¥29-49/月（无限验证 + GEO 深度分析 + 交叉验证）
• 目标：3-5% 付费转化率 → 月收入 ¥5-10 万
• 对标 Facticity.AI 的 $9.99/月定价

阶段 3：B 端 + 数据变现（12 个月+）
• API 服务：媒体/企业按量接入
• GEO 洞察报告：出售给 AI 平台/广告监管/媒体
• 这是真正的利润池

八、最终结论

值得做。核心理由：

1. 技术壁垒低（好事）→ 你可以快速出产品
2. 市场壁垒低（坏事）→ 你需要用数据积累构建护城河
3. 时机精准 → AI 信任焦虑正在上升，但用户行为还未被竞品锁定
4. 国内空白 → 你是第一个吃螃蟹的，先发至少有 6-12 个月窗口

最大的不确定性：C 端用户是否愿意为"AI 答案可信度"付费？
唯一的验证方式：先做出来，让市场回答。

建议：投入 3-5 天做 MVP → 冷启动 → 看数据 → 再决定是否全力投入。
